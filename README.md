# SyntaxWear

SyntaxWear é uma landing page estática de um e-commerce de tênis e sneakers, com foco em identidade visual moderna, navegação clara e uma experiência responsiva.

## Objetivo do projeto

Este projeto foi criado para apresentar uma loja fictícia de moda urbana com:
- header fixo e navegação organizada;
- seção hero com destaque principal;
- categorias de produtos;
- grid visual com cards de destaque;
- footer com newsletter, redes sociais e navegação;
- layout responsivo para diferentes tamanhos de tela.

## Tecnologias utilizadas

- HTML5
- CSS3
- Arquivos de imagem e ícones em SVG

## Como executar localmente

1. Baixe ou clone o projeto.
2. Abra a pasta no seu editor de código.
3. Rode o arquivo [index.html](index.html) diretamente no navegador ou use uma extensão como Live Server no VS Code.

## Como personalizar

- Texto e conteúdo: edite [index.html](index.html)
- Cores, tipografia e variáveis: veja [css/variables.css](css/variables.css)
- Estilos do layout geral: [css/base.css](css/base.css)
- Estilos responsivos: [css/responsive.css](css/responsive.css)
- Componentes específicos: [css/components](css/components)
- Imagens e ícones: [assets/images](assets/images)
